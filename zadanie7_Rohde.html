1. Na co pozwala styl lokalny, który element będzie formatowany z użyciem stylu lokalnego?

Styl lokalny pozwala na formatowanie pojedynczego, konkretnego elementu w kodzie HTML. Formatowanie dotyczy tylko tego elementu, w którym styl jest zdefiniowany.

2. Składnia teoretyczna (z użyciem ramki otaczającej→ może być tabela o jednej komórce, zastosuj „znaki specjalne” do napisania definicji stylu lokalnego tak aby instrukcja CSS nie wykonywała się a tylko wyświetlała wewnątrz ramki [ spacja niełamana] przykład „znaku specjalnego”)

<element style="cecha: wartość; cecha2: wartość2;">

3. Co może być selektorem?

W stylu lokalnym selektorem jest sam element HTML, do którego dodajemy atrybut style. Nie używamy dodatkowych selektorów, jak w arkuszach CSS.

4. Co wipisujemy jako "cechę". Jakie jest znaczenie średników?

Jako "cechę" wpisujemy właściwość CSS, którą chcemy zmienić (np. color, font-size, background-color). Średniki oddzielają poszczególne deklaracje stylów (cecha: wartość).

5. Co wipisujemy jako "wartość" (w deklaracji stylu)?

Jako "wartość" wpisujemy konkretną wartość właściwości CSS (np. red, 16px, #FFFFFF). Wartość musi być zgodna z typem danej właściwości.
