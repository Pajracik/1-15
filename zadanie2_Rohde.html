CSS (Cascading Style Sheets) –u>
to język arkuszy stylów używany do opisu wyglądu i formatowania dokumentów HTML oraz XML. CSS pozwala na kontrolowanie układu strony, kolorów, czcionek oraz innych elementów wizualnych, co umożliwia tworzenie atrakcyjnych i spójnych stron internetowych. Dzięki CSS można oddzielić strukturę dokumentu (HTML) od jego prezentacji, co ułatwia zarządzanie stylem i zwiększa elastyczność projektowania stron.
